O que são Media Queries? R: A responsividade em telas pequenas, por exemplo, faz com que o site seja mais ou menos parecido com o que temos na tela grande. Dispositivos mobiles são prioridades para a responsividade. (Mobile First) Ex1: @media (max-width: 600px) { body { } } Ex2: @media (min-width: 600px) { body { } }
